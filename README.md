🧠 Mental Stress Detector

📖 Project Description

Mental Stress Detector is a Machine Learning based web application that predicts whether a person is experiencing mental stress based on textual input.

The system analyzes user-provided text and classifies it using a trained Machine Learning model.

This project was developed using Python and implemented in Jupyter Lab for model development and training.

💻 Technologies Used

1.Python

2.Machine Learning (Scikit-learn)

3.Pandas

4.NumPy

5.Jupyter Lab

6.Streamlit

🤖 Machine Learning Workflow

The project follows these steps:

. Data Collection

. Data Preprocessing

.Feature Extraction

. Model Training

.Model Evaluation

📊 Model Details

1. Input: Text (user’s sentence or message)

2. Output: Stress / No Stress prediction

3. Text is converted into numerical features using TF-IDF

4. A trained ML model is used to classify the input

🚀 How to Run the Project

▶️ If Running in Jupyter Lab:

1. Open Jupyter Lab.

2. Open the .ipynb notebook file.

3. Run all cells step by step.

▶️ If Running as a Web App (Streamlit):

1. Open terminal in project folder.

2. Run:

#Code

streamlit run app.py

3. The application will open in your browser.
